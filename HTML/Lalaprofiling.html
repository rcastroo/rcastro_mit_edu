Summary
Lalaprofiling is a graphical application that facilitates the creation of paths and motion profiles for robot autonomous routines, along with separate classes on the controller side to follow the desired profiles.
The problem with current solutions is that they are too hard to implement. A regular team with limited knowledge and resources cannot successfully implement complex autonomous routines.So I developed Lalaprofiling on my free time as a tool for my FRC team, along with other ones I could work with to implement. In 2017, I implemented it with my team, CRUSH 1011, and then in 2018, I used it with a team I helped mentor, team 238 The Crusaders.

You can find further documentation here: Documentation
Goals
The design goals coming for Lalaprofiling were set as follows:
  • Main design goal: Has to be easy enough to implement and use that anyone with minor coding experience can implement in their team with limited help.
  • Must be able to generate S-curve profiles
  • Must be capable of moving a robot on a 90 in circular diameter path 90 degrees with less than an inch of error in both the x and y directions
Current State
After testing the program with a different team, team 238, I found that it was more difficult to implement and tune than I imagined. More work needs to be done in order to make it more accessible for others. In the coming years, I will be working on making this a reality for public teams.
